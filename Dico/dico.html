Le Lexique :

  • L'alimentation : Ce composant est surement l'un des plus importants, ce dernier permet l'alimentation en énergie de tous vos composants.
  • La carte mere OU MB : L'élément crucial de votre ordinateur, c'est celui qui relie tous les composants entre eux pour n'en former qu'un.
  • Le processeur : Sans lui rien n'est possible, le processeur ou CPU s'occupe de tous les calculs de la machine.
  • Le disque dur : Il permet le stockage de toutes vos données, il peut être appellé HDD ou SSD (deux technologies différentes).
  • La carte graphique : Cette carte qui se branche, la plupart du temps en PCI-E, permet l'affichage sur votre écran et gère toute la partie graphique de votre ordinateur.
  • Le BIOS : C'est le systeme permettant le bon démarage de votre ordinateur et son bon fonctionnement.
  • La mémoire RAM : c'est une mémoire temporaire pouvant stocker les données générés par l'ordinateur, elles ont une lecture/écriture très rapide.
  • ATX : C'est le format le plus répandu de carte mère, il existe (par ordre de grandeur décroissant): L'E-ATX, ATX, Mini-ATX, Micro-ATX, flex-ATX, mini-ITX.
  • PCI-E : c'est la technologie la plus répandue pour toute type de cartes externes : carte graphique, carte son, carte réseau...
  • Socket : C'est le récéptacle du processeur, il peut varier en fonction des marques et des modèles.
  • Driver : C'est un logiciel qui permet au systeme d'exploitation de bien savoir utilisé le matériel de votre ordinateur.
